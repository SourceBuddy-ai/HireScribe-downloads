

# HireScribe Downloads

Privacy-first interview transcription for recruiters. Record, transcribe, and summarize interviews with 100% local AI processing.


## About HireScribe

HireScribe is a desktop application built specifically for recruiters who want to focus on conversations instead of note-taking. All processing happens locally on your machine - your candidate data never leaves your computer.

### Key Features

- Local transcription using Whisper AI
- AI-powered interview summaries
- Zero cloud dependencies
- GDPR compliant by design
- Works offline after initial setup

### System Requirements

**Windows**
- Windows 10 (64-bit) or newer
- 8GB RAM minimum (16GB recommended)
- 4GB free disk space
- Intel Core i5 or AMD equivalent (2015 or newer)

**macOS**
- macOS 11 (Big Sur) or newer
- 8GB RAM minimum (16GB recommended)
- 4GB free disk space
- Apple Silicon (M1/M2/M3)
